# GlaucoTrack
GlaucoTrack – an intelligent eye health screening system

GlaucoTrack is an AI-based eye health screening system designed to assist in the early detection of glaucoma. The system analyzes eye images and provides risk predictions, helping users take preventive action before the condition becomes severe.

The application is built using Python for the backend, with HTML and CSS for the frontend, and integrates a database (MySQL/PostgreSQL) to store user data and screening results. It focuses on improving accessibility to eye health monitoring by offering a simple and efficient digital solution.

Key features include real-time analysis, user-friendly interface, and secure storage of medical data. The system aims to support both individuals and healthcare providers in early diagnosis and awareness.
